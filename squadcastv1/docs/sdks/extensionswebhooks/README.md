# ExtensionsWebhooks
(*extensions.webhooks*)

## Overview

### Available Operations

* [get_all](#get_all) - Get All Webhooks

## get_all

Returns all the webhooks of the organization.
Requires `access_token` as a `Bearer {{token}}` in the `Authorization` header with `read` scope.

### Example Usage

<!-- UsageSnippet language="python" operationID="Webhooks_getAllWebhooks" method="get" path="/v3/extensions/event-webhooks" -->
```python
from squadcast_sdk import SquadcastSDK


with SquadcastSDK(
    bearer_auth="<YOUR_BEARER_TOKEN_HERE>",
) as ss_client:

    res = ss_client.extensions.webhooks.get_all()

    # Handle response
    print(res)

```
